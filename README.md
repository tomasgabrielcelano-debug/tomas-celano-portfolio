# Tomas Celano Portfolio

Personal portfolio built with React, TypeScript and Vite.

## Featured project

RepairShop Management System: full-stack internal management system for repair shops.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Cloudflare Pages

- Framework preset: Vite
- Build command: `npm run build`
- Output directory: `dist`
